História do JavaScript

JavaScript foi criado em 1995 por Brendan Eich na Netscape, inicialmente chamado de Mocha. O nome foi mudado para LiveScript e, por fim, JavaScript, aproveitando a popularidade do Java.
A linguagem foi projetada para tornar as páginas web interativas, permitindo validações de formulários e animações.
Nos anos 2000, ganhou destaque com bibliotecas como jQuery e a introdução do AJAX, revolucionando a experiência do usuário na web.
Atualmente, é uma das principais linguagens para desenvolvimento web, sendo usada em frontend e backend com o Node.js.
